# GalacticraftTools

这是一个解决星系附属模组兼容性的工具。

## 使用

1.在[Releases](https://github.com/FuBangkun/GalacticraftTools/releases)下载最新版本的程序。

2.安装你要玩的星系附属，并启动游戏，以生成配置文件（必须启动，不然没有配置文件无法使用本工具）

3.将可执行文件移至config文件夹内，双击打开。

4.按照你的实际情况进行选择，然后确认修改即可。

## 已知问题

1.因为太阳系模组本身注册系统的原因，可能会导致崩溃，本工具只会修改太阳系模组的星图，无法解决该问题。
